# 탐욕
- 탐욕(Greedy) 알고리즘은 현시점에서 가장 적합하다고 생각되는 요소를 선택해가는, 즉 근시안적 선택을 수행하는 알고리즘 기법
- 데이터 사이의 관계 혹은 앞서 선택한 요소의 영향 등을 고려하지 않아도 됨
- 최적화 문제(optimization problem)를 해결하는데 사용할 수 있는 가장 간단하며 쉽게 구현할 수 있는 알고리즘
- i번째 단계에서 A라는 조건을 이용하여 최적의 해를 선택했다면, i번째 이외의 모든 단계에서도 A라는 조건을 이용하여 해를 구해야 함
- 코딩테스트와 같이 빨리 문제를 풀어야 하는 상황이 아니라면 한번쯤은 탐욕 알고리즘을 이용하여 짠 코드가 왜 정답인지 확인해보는 것도 좋은 공부가 됨
- 거스름돈 거슬러주기, 최소비용 신장 트리
